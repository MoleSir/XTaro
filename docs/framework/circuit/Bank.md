# Bitcell Array

输入配置的是地址位数 AW 与数据位数 DW。那么理论上 Bitcell 阵列的大小为：2^AW * DW

但是通常 Bitcell 阵列都不会按照这个大小分配。通常会使用列选择器，将地址的部分转移到列地址。使得译码电路更容易实现，并且版图空间利用更合理。

而实际上列选择器也需要使用译码电路（译码器 + 三态门）。所以列地址不宜过大，如果只使用 1-2、2-4、3-8 与 4-16 译码的情况下。
